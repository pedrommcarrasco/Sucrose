## Instalation üì¶

Sucrose is available through [CocoaPods](https://cocoapods.org/pods/Sucrose). To do so, add the following line to your Podfile:

```swift
pod 'Sucrose'
```
And then run the following command in terminal:

```swift
pod install
```
